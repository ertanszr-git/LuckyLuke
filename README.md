# LuckyLuke

LuckyLuke is a script for scanning systems or websites and finding vulnerabilities very easy.

## Installation

You have to use Debian based distributions for this script.

```bash
python3 install.py
sudo bash exe.sh
```

## Usage

```bash
luckyluke
```
